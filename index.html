世界观设定 · 银蔷薇典章

银蔷薇典章
Codex of the Silver Rose
当铠甲成为礼服，当利剑即是礼仪——贵族的血液流淌着毁灭的优雅
银蔷薇典章的舞台——「银质纪元」的大陆阿尔真蒂亚（Argentia），是一个以「契约灵魂」为核心驱动力的贵族文明世界。在这里，战争是最高形式的礼仪，铠甲是最精密的时装，每一位持剑者都是这个世界秩序的具现。
世界的能量来源于「铸灵契约」——人类与某种古老意志缔结灵魂盟约，以此获得铸造「活甲」的能力。活甲并非单纯的金属装备，而是承载着持有者灵魂共振的有机战甲，会随主人的情感与意志不断演化。契约等级越深，铠甲的纹饰便越趋近艺术，而非武器。
阿尔真蒂亚由五大贵族家族共同执掌，各家掌握着不同的铸灵传承与战斗美学。百年和约在一场名为「玫瑰审判」的神秘事件后彻底崩溃——有人开始掌握让活甲「吞噬」宿主灵魂的禁术，整个贵族秩序面临从内部瓦解的危机。一切纷争通过高度仪式化的「剑礼」制度解决——胜者不一定要杀死对手，但必须使对方的活甲永久失去某一片区域的纹饰，这意味着一部分灵魂契约的永久封印。
视觉语言建立在「高贵的暴力」与「精致的毁灭」之间的张力上。这是一个以「美即力量」为信仰的世界——铸灵师的精神境界越高，其铸造出的活甲便越精美繁复。花朵纹饰代表生命力，蕾丝代表对死亡的轻盈态度，羽毛象征灵魂的自由度量。银、白、黑、金四色为主调，最可怕的敌人往往穿着最繁复的铠甲。
Aesthetic Code
美学语言
六大核心美学支柱：巴洛克铠甲美学——所有战甲以过度繁复的纹饰为荣，将蔷薇、蔓藤、牡丹等植物纹样精密铸刻于金属之上；东西方文化混融——旗袍、汉服与欧式礼服共存，武士刀与细剑并列出现；极致色彩克制——银白黑金四色为主，粉色代表感情流露，红色代表危险与禁忌；冷静的高强度戏剧性——内敛克制的表情与充满戏剧张力的姿态形成反差；时装逻辑的战斗设计——高跟鞋是战斗标配，蕾丝裙摆提供视觉干扰，美丽本身即是战术。
